Meet the team behind the Plant Stress Classification System

Semester Project — Machine Learning & Data Science

A Web-Based Plant Stress Level Classification System
Using Machine Learning Based on Biosensor Data and Chlorophyll Content

This project implements three machine learning algorithms — SVM, KNN, and Neural Network (MLP) — to classify plant health status from biosensor data. The models are trained with hyperparameter optimization using GridSearchCV and deployed through a modern Serverless API via Vercel.
